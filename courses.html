Computer Science 61A (Spring 2019)
CS61A focuses on the idea of abstraction and primarily uses the Python3 programming language. Besides learning how to code in python, I also gained a lot of knowledge regarding the conceptual aspect of coding. One of my favorite parts of the class would be the environmental diagrams because I not only found them fun, but I also found them useful in understanding how a computer reads code and how I can be more efficient on my end. You can view the course website here for more information regarding the course.

Data 8 (Spring 2019)
DATA8 is Berkeley's introductory data science course. I was introduced to various aspects of data analysis and practiced those concepts through weekly homeworks and labs. The projects in the course were long but definitely improved my skills as a data analyst and a programmer. You can view the course website here for more information regarding the course.

Fundamentals of Programming (Summer 2019)
I took this course through my local community college over summer for fun. It is an introductory course that teaches about the fundamentals of game programming using C++. Although about programming games, the biggest takeaway for me was learning a new programming language and being able to apply that new knowledge.
